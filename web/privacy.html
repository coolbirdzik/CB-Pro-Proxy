Privacy Policy

Effective Date: January 1, 2026

This Privacy Policy applies to the CB Pro Proxy mobile application (the "Application") provided by Hung Nguyen Tan (the "Service Provider"). The Application is provided "AS IS".

Information Collection and Use

CB Pro Proxy is a device-level proxy client that uses Android VpnService to route network traffic through a proxy endpoint that you configure. The Service Provider does not operate those proxy servers unless explicitly stated.

  • Proxy profiles (host, port, protocol, optional username) are stored locally on your device to provide the service.
  • Proxy credentials (password) are stored using the platform secure storage (Keychain on iOS / Keystore-backed storage on Android) when provided.
  • Local diagnostics (optional): the Application may store connection status and troubleshooting logs locally on your device to help you debug connectivity issues. You can delete these logs by clearing the Application data or using in-app controls if available.

The Application does not require you to create an account.

The Application does not collect precise location data.

The Service Provider does not sell your data.

Third Party Access

Network traffic routed through the Application is sent to the proxy server you configure. That proxy server may be operated by a third party (for example, an enterprise proxy or a proxy vendor) and will be able to observe traffic that it receives according to its own policies.

The Application may contact third-party endpoints to provide optional features such as public IP checks or network speed tests (for example, services similar to ipify, ip-api, or Cloudflare speed test). These requests originate from your device (directly or via your configured proxy, depending on your settings).

Opt-Out Rights

You can stop all processing by uninstalling the Application.

Data Retention Policy

Proxy profiles, settings, and diagnostics are stored on your device. They remain until you delete them, clear the Application data, or uninstall the Application.

Children

The Application is not directed to children under the age of 13. The Service Provider does not knowingly collect personal information from children.

If you believe a child has provided personal information to the Service Provider, please contact the Service Provider so appropriate action can be taken.

Security

The Application stores proxy credentials using secure storage provided by the platform. You are responsible for keeping your device secure and for choosing trusted proxy endpoints.

You are advised to consult this Privacy Policy regularly for any changes, as continued use is deemed approval of all changes.
